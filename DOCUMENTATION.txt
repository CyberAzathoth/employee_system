SECURITY MEASURE #1: PASSWORD HASHING
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€
Technology: Bcrypt (via PASSWORD_DEFAULT)
Implementation: password_hash() and password_verify()

How it works:
- Plain text passwords NEVER stored
- Each password gets unique random salt
- Cost factor of 10 (1,024 iterations)
- 60-character hash generated
- One-way encryption (cannot be reversed)

Example:
Input: "password123"
Hash: $2y$10$92IXUNpkjO0rOQ5byMi.Ye4oKoEa3Ro9llC/.og/at2.uheWG/igi

SECURITY MEASURE #2: SQL INJECTION PREVENTION
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€
Technology: PDO Prepared Statements
Implementation: prepare() + bindParam() + execute()

How it works:
- SQL structure separated from data
- User input treated as data only
- Special characters automatically escaped
- Database-level protection

Vulnerable: "SELECT * FROM User WHERE username = '$username'"
Secure: prepare("SELECT * FROM User WHERE username = :username")

1. LOGIN
   - Go to login page
   - Enter username and password
   - Click Login
   - Redirected to Dashboard

2. MANAGE EMPLOYEES (All Roles)
   - Click "Employees" in sidebar
   - Fill form to add new employee
   - Click "Edit" button to modify
   - Click "Delete" (Admin/Super Admin only)

3. MANAGE USERS (Admin/Super Admin)
   - Click "Users" in sidebar
   - Select employee from dropdown
   - Enter username and password
   - Choose role
   - Click "Create User"

4. CHANGE PASSWORD (Admin/Super Admin)
   - Go to Users page
   - Click "Change Password" button
   - Enter new password
   - Submit

5. VIEW PASSWORD LOGS (All Roles)
   - Click "Password Logs" in sidebar
   - Switch between tabs:
     * Password Changes
     * Login Attempts

6. VIEW DELETED EMPLOYEES (Admin/Super Admin)
   - Click "Deleted Employees" in sidebar
   - View log of deleted records
   - No restore available

7. VIEW ACTIVITY LOGS (Admin/Super Admin)
   - Click "Activity Logs" in sidebar
   - See all user actions
   - Filter by date/user/module

8. LOGOUT
   - Click "Logout" button in header
   - Redirected to login page
